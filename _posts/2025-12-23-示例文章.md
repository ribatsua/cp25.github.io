# Markdown语法入门与实践

欢迎来到我的个人知识库！这是一篇关于Markdown语法的示例文章。

<!-- more -->

## 一、什么是Markdown？

Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML（或HTML）文档。

## 二、基本语法

### 1. 标题

使用`#`号可以表示1-6级标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 2. 段落与换行

段落之间使用空行分隔。如果需要在段落内换行，可以在行尾添加两个或多个空格，然后回车。

### 3. 强调

```markdown
*斜体文本*
**粗体文本**
***粗斜体文本***
```

*斜体文本*
**粗体文本**
***粗斜体文本***

### 4. 列表

#### 无序列表

使用`*`、`+`或`-`作为列表标记：

```markdown
* 列表项1
* 列表项2
  * 子列表项2.1
  * 子列表项2.2
```

* 列表项1
* 列表项2
  * 子列表项2.1
  * 子列表项2.2

#### 有序列表

使用数字加`.`作为列表标记：

```markdown
1. 第一步
2. 第二步
   1. 子步骤2.1
   2. 子步骤2.2
```

1. 第一步
2. 第二步
   1. 子步骤2.1
   2. 子步骤2.2

### 5. 代码

#### 行内代码

使用反引号(`)包裹代码：

```markdown
使用`npm install`命令安装依赖。
```

使用`npm install`命令安装依赖。

#### 代码块

使用三个反引号(```)包裹代码块，并可以指定语言：

```javascript
function hello() {
    console.log("Hello, World!");
}

hello();
```

```python
def hello():
    print("Hello, World!")

hello()
```

### 6. 链接

```markdown
[链接文本](链接地址 "链接标题")
```

[GitHub](https://github.com "GitHub官网")

### 7. 图片

```markdown
![图片描述](图片地址 "图片标题")
```

![示例图片](https://picsum.photos/800/400 "这是一张示例图片")

### 8. 表格

```markdown
| 表头1 | 表头2 | 表头3 |
|------|------|------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

| 表头1 | 表头2 | 表头3 |
|------|------|------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

## 三、高级用法

### 1. 引用

```markdown
> 这是一段引用文本。
> 
> > 这是嵌套引用。
```

> 这是一段引用文本。
> 
> > 这是嵌套引用。

### 2. 分割线

使用三个或更多的`*`、`-`或`_`：

```markdown
***
---
___
```

***

### 3. 任务列表

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 待处理的任务
```

- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 待处理的任务

## 四、总结

Markdown是一种简单易用的标记语言，非常适合用于写作、文档编写和内容创作。通过掌握这些基本语法，你可以轻松地创建格式化的文档。
